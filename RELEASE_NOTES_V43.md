# 🚀 Release Notes — Version 43.0.0

**Дата релиза:** 27 марта 2026  
**Статус:** ✅ Production Ready  
**Тип релиза:** Major — Стабилизация для Xcode и GitHub

## 📋 Обзор

Версия 43.0.0 представляет собой стабильную production-ready сборку приложения **Babkak Summer 2026**, полностью подготовленную для деплоя в GitHub и сборки нативного iOS-приложения в Xcode. Главное изменение — намеренное удаление функционала Push-уведомлений из-за неразрешимых конфликтов зависимостей.

---

## ⚠️ Breaking Changes

### Удален функционал Push-уведомлений
- **Причина:** Неразрешимые конфликты зависимостей пакета `@capacitor/push-notifications`
- **Действие:** Полное удаление пакета и всей связанной логики
- **Влияние:** Приложение больше не отправляет и не получает push-уведомления

**Что это значит для пользователей:**
- ❌ Нет push-уведомлений о новых событиях
- ❌ Нет напоминаний через уведомления
- ✅ Все остальные функции работают в полном объеме
- ✅ Приложение стабильнее и проще в поддержке
