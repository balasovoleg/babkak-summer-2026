## 🎨 Ключевые особенности

### Минимализм:
- Меньше элементов
- Больше пространства
- Фокус на контенте

### Glassmorphism 2.0:
```css
background: rgba(255,255,255,0.5)
backdrop-filter: blur(20px)
border: 1px solid rgba(255,255,255,0.6)
box-shadow: 0 8px 32px rgba(0,0,0,0.08)
```

### Градиенты:
```css
/* Заголовки */
linear-gradient(135deg, #7c3aed, #ec4899, #3b82f6)

/* Кнопки */
linear-gradient(135deg, #7c3aed, #ec4899)

/* Фоны */
linear-gradient(135deg, 
  rgba(250,240,255,0.95),
  rgba(255,245,250,0.92),
  rgba(245,250,255,0.95)
)
```

### Свайпы:
- 👈 Влево = Следующий месяц
- 👉 Вправо = Предыдущий месяц
- 🖱️ Работает с мышью
- 📱 Работает на тачскринах

## 🚀 Новые возможности

### ✅ Свайп-навигация
- Работает на desktop (с мышью!)

- Минимум 50px для срабатывания
- Плавные переходы

- Первый месяц → кнопка "Главная"

### ✅ Подсказка о свайпах
- Показывается 1 раз
- Автоматически исчезает через 3 сек
- Сохраняется в sessionStorage

### ✅ Floating install button
- Вверху справа
- Не отвлекает
- Компактный
- Gradient дизайн

## 📱 Адаптивность

Все обновления полностью responsive:

### Mobile (< 640px):
- Текст: text-5xl → text-6xl
- Padding: p-6
- Grid: 1 колонка
- Навигация: только иконки

### Tablet (640-1024px):
- Текст: text-6xl
- Padding: p-8
- Grid: 1 колонка
- Навигация: иконки + текст

### Desktop (≥ 1024px):
- Текст: text-7xl → text-8xl
- Padding: p-12
- Grid: 2 колонки
- Навигация: полная

## 💡 Рекомендации для использования

### Навигация:
1. **Кнопки** — нажмите на стрелки
2. **Свайпы** — проведите влево/вправо
3. **Keyboard** (скоро) — стрелки ← →

### Установка:
1. Нажмите кнопку "Установить" (вверху справа)
2. Следуйте инструкциям
3. Иконка появится на экране

### Locket Widget:
1. Нажмите "💖 Локет виджет" на Cover
2. Загрузите фото в галереи месяцев
3. Они автоматически появятся в виджете
