### 2. 🎨 Генератор иконок из группового фото

Создан **HTML генератор** для автоматического создания иконок приложения!

**Файл:** `/public/icons/generate-app-icons.html`

#### Особенности:

- 📸 **Загружаете** групповое фото
- 🤖 **Автоматически** создаются 8 размеров
- 🎨 **Красивый дизайн:**
  - Пастельный градиент фон (розовый → фиолетовый → голубой)
  - Закругленные углы (15%)
  - Padding 10% вокруг фото
  - Автоматическое масштабирование (fit)
  - Легкая тень для объёма
- 📥 **Скачать все** одной кнопкой

#### Размеры иконок:

1. `icon-72x72.png` - iOS Safari
2. `icon-96x96.png` - Android Chrome
3. `icon-128x128.png` - Chrome Desktop
4. `icon-144x144.png` - Windows Tiles
5. `icon-152x152.png` - iOS Safari (iPad)
6. `icon-192x192.png` - **Android основная**
7. `icon-384x384.png` - Android (high density)
8. `icon-512x512.png` - **Основная PWA**, splash screen

## 🚀 Как использовать

### Шаг 1: Сгенерировать иконки

1. **Откройте генератор:**
   ```
   http://localhost:5173/icons/generate-app-icons.html
   ```

2. **Загрузите фото:**
   - Нажмите "📸 Загрузить фото"
   - Выберите групповое фото с главной страницы
   - (Оно импортировано как `figma:asset/625d745ab89a842ea88976a737d1b12372f4c718.png`)

3. **Скачайте:**
   - Нажмите "📥 Скачать все иконки"
   - Автоматически скачаются 8 PNG файлов

4. **Установите:**
   - Поместите все PNG файлы в `/public/icons/`
   - Готово! 🎉

### Шаг 2: Проверить

**DevTools (F12):**
```
Application → Manifest → Icons
```

Все 8 иконок должны загружаться без ошибок ✅

**На устройстве:**
1. Нажмите "Установить" (кнопка внизу слева)
2. Или в Safari: Поделиться → "На экран «Домой»"
3. Проверьте иконку - должно быть групповое фото! 🌸

## 💡 Советы

### Для лучшей иконки:

- ✅ Используйте **квадратное фото** (или близко к квадрату)
- ✅ Минимальное разрешение: **512×512px**
- ✅ **Яркие цвета** лучше видны
- ✅ **Центрируйте** важные элементы
- ❌ Избегайте мелкого текста
- ❌ Избегайте очень темных фото

### Для кнопки Locket:

- 💖 Кнопка под фото - логичное место
- ✨ Розовый цвет ассоциируется с Locket
- 🎨 Выделяется на фоне фиолетовой темы
- 📱 Открытие в новой вкладке - удобно
