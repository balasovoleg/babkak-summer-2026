## 🚀 Как протестировать?

### 1. Локально (Development)
```bash
npm run dev
# или
vite
```

Откройте в Chrome и проверьте:
- DevTools → Application → Service Workers
- DevTools → Application → Manifest
- Lighthouse → PWA аудит

### 2. Production (Обязательно HTTPS!)

PWA требует HTTPS для работы (кроме localhost).

Разверните на:
- Vercel (рекомендуется)
- Netlify
- GitHub Pages
- Или любой хостинг с SSL

## 📊 Проверьте PWA Score

1. Откройте DevTools (F12)
2. Вкладка "Lighthouse"
3. Выберите "Progressive Web App"
4. Нажмите "Analyze"

**Цель: 90+ баллов**

## 🎨 Последний шаг: Иконки

### Быстрая генерация:

1. Откройте в браузере:
   ```
   http://localhost:5173/icons/generate-icons.html
   ```

2. Нажмите "Скачать все"

3. Сохраните файлы в `/public/icons/`

**Или** используйте онлайн-сервис:
- [PWA Builder](https://www.pwabuilder.com/imageGenerator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)

См. `/public/icons/QUICK_SETUP.md` для деталей.

```
/public/
  ├── manifest.json          # Манифест приложения ✅

  └── PWAInstall.tsx         # Промпт установки ✅

/index.html                  # Обновлён с meta-тегами ✅
/src/main.tsx               # Точка входа ✅
/vite.config.ts             # Настроен для PWA ✅
```

## 🔧 Настройки

### Изменить информацию приложения:
Отредактируйте `/public/manifest.json`:
```json
{
  "name": "Ваше название",
  "short_name": "Короткое",
  "description": "Описание",
  "theme_color": "#f8b4d9"
}
```

### Настроить кэширование:
Отредактируйте `/public/sw.js`:
- Измените `CACHE_NAME` для принудительного обновления
- Добавьте файлы в `PRECACHE_URLS`
