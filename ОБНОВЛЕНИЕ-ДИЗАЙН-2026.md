**Добавили:**

### 4. 🔘 **Интегрированная кнопка "Установить"**

**Было:** Внизу слева, большая
```

```

**Стало:** Вверху справа, компактная
```
[↓ Установить] ← Floating, минималистичная
```

**Особенности:**
- Gradient иконка с blur эффектом
- Gradient текст (purple→pink)
- Белый фон с backdrop-blur
- Не отвлекает от контента

### 5. 🎨 **Современные фоны**

Новые градиенты с высоким blur:

```css

  rgba(250,240,255,0.95), /* Lavender */

  rgba(245,250,255,0.95)  /* Blue */
)
backdrop-filter: blur(40px)
```

**Эффект:**
- Фото видно через прозрачный overlay
- Мягкий, воздушный, совреме��ный
- Приятная эстетика

## 💡 Советы по использованию

### 1. Генерация иконок

Откройте `/icons/generate-app-icons.html` и создайте иконки из группового фото:

```
http://localhost:5173/icons/generate-app-icons.html
```

### 2. Установка на iPhone

1. Нажмите кнопку "Установить" (вверху справа)
2. Следуйте инструкциям
3. Иконка с вашим фото появится на экране!

### 3. Locket Widget

Нажмите "💖 Локет виджет" под фото на главной странице

## 🌟 Примеры применения

### Заголовок с gradient:

```tsx
<h1
  style={{ 
    fontFamily: "var(--font-display)",
    background: "linear-gradient(135deg, #7c3aed, #ec4899, #3b82f6)",
    WebkitBackgroundClip: "text",
    WebkitTextFillColor: "transparent",
    backgroundClip: "text"
  }}
>
  Babkak 2026
</h1>
```

### Pill badge:

```tsx
<div className="inline-flex items-center gap-2 px-4 py-2 rounded-full bg-white/60">
  <span 
    style={{ 
      color: "#9b6dd6",
      fontFamily: "var(--font-display)"
    }}
  >
    ЛЕТО 2026
  </span>
</div>
```

### Glassmorphism карточка:

```tsx
<div
  style={{
    background: "rgba(255,255,255,0.5)",
    backdropFilter: "blur(20px)",
    border: "1px solid rgba(255,255,255,0.6)",
    boxShadow: "0 8px 32px rgba(0,0,0,0.08)"
  }}
>
  {/* Контент */}
</div>
```
