### 2. 🔄 **Свайп-навигация**

Добавлена интуитивная свайп-навигация для быстрого перемещения между месяцами:

**Как работает:**
- 👈 **Свайп влево** → Следующий месяц
- 👉 **Свайп вправо** → Предыдущий месяц
- 🖱️ **Работает с мышью** — можно тестировать на desktop
- 📱 **Работает на тачскринах** — идеально для мобильных

**Технические детали:**
```tsx
import { useSwipeable } from "react-swipeable";

const handlers = useSwipeable({
  onSwipedLeft: () => navigate(nextPath),
  onSwipedRight: () => navigate(prevPath),
  trackMouse: true,        // Тестирование мышью
  preventScrollOnSwipe: false,
  delta: 50,               // Минимум 50px для свайпа
});
```

## 📁 Новые компоненты

### 1. `MonthNavigation.tsx`

Универсальный компонент навигации для всех месяцев:

```tsx
<MonthNavigation
  prevMonth={{ path: "/may", label: "Май" }}
  nextMonth={{ path: "/july", label: "Июль" }}
  currentMonth="июнь"
  color={color}
/>
```

**Props:**
- `prevMonth?` — предыдущий месяц (или null)
- `nextMonth?` — следующий месяц (или null)
- `currentMonth` — название текущего месяца
- `color` — цвет темы месяца

### 2. `SwipeableMonth.tsx`

HOC для добавления свайп-функциональности:

```tsx
<SwipeableMonth prevPath="/may" nextPath="/july">
  {children}
</SwipeableMonth>
```

**Props:**
- `prevPath?` — путь к предыдущей странице
- `nextPath?` — путь к следующей странице
- `children` — содержимое страницы

### 3. `SwipeHint.tsx`

Подсказка для пользователей о свайп-навигации:

```tsx
<SwipeHint />
```

**Автоматически:**
- Показывается 1 раз
- Исчезает через 3 секунды
- Сохраняет состояние в sessionStorage

## 🚀 Как использовать

### Навигация между месяцами:

**1. Кнопки:**
- Нажмите на стрелки в навигации
- Кнопка "Главная" → возврат на Cover
- Кнопка "Rewind" → переход к ретроспективе

**2. Свайпы (desktop и mobile):**
- Свайп влево → следующий месяц
- Свайп вправо → предыдущий месяц
- Работает с мышью на desktop!

**3. Клавиатура (скоро):**
- ← → стрелки для навигации
- Esc → возврат на главную

### Пример свайпа мышью:
1. Зажмите левую кнопку мыши
2. Проведите влево или вправо (минимум 50px)
3. Отпустите кнопку
4. Автоматический переход!
