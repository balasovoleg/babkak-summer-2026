## Всего 4 команды для запуска в Xcode:

```bash
# 1. Установите зависимости
npm install

# 2. Соберите проект
npm run build

# 3. Синхронизируйте с iOS (вместо cap add ios если уже добавлено)
npm run cap:sync

# 4. Откройте в Xcode
npm run cap:open:ios
```

## 📱 В Xcode:

1. Откроется файл: `ios/App/App.xcworkspace`
2. **Signing & Capabilities** → выберите ваш Team
3. **General** → установите Version: `43.0.0`, Build: `43`
4. Выберите симулятор iPhone или реальное устройство
5. Нажмите ▶ **Run** (или Cmd + R)
6. Приложение запустится на выбранном устройстве!
